# normalize

Prosta aplikacja command-line napisana w języku C, która normalizuje
podane liczby do przedziału [0,1].

## Instalacja

1. Sklonuj repozytorium:

```bash
git clone https://github.com/TWOJ_USERNAME/TWOJE_REPO.git
cd TWOJE_REPO

2.Uruchom skrypt instalacyjny:

chmod +x installer.sh
./installer.sh

3.Dodaj ścieżkę do zmiennej PATH (zgodnie z instrukcją wyświetloną przez instalator).

##Użycie

normalize x1 x2 x3 ...
